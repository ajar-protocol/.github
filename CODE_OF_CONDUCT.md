# Code of Conduct

The Ajar Protocol community adopts the Contributor Covenant v2.1 (https://www.contributor-covenant.org/version/2/1/code_of_conduct/) across all repositories in this organization.

Report violations privately to the maintainers (conduct@ajarprotocol.org once live; until then, via a private message to org owners). Enforcement follows the Covenant's guidelines. Be specific, be kind, argue about ideas by citing documents — never about people.
